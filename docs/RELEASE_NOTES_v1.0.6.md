# Release Notes - v1.0.6

## 🇷🇺 Русский

### Что нового

- Добавлено подробное русское руководство по аддону
- Добавлено подробное английское руководство по аддону
- На главной странице проекта появились ссылки на оба руководства
- Улучшена структура документации для GitHub
- Исправлено поведение `TO FLOOR` - теперь объект корректно ставится нижней частью на пол

### Основные возможности версии

- `ALIGN` для ручного выравнивания по сохранённым сторонам
- `AUTO ALIGN` для автоматической укладки модели
- `TO FLOOR` для опускания объекта на `Z = 0`
- `FLIP` для переворота результата на `180°`
- `Quick Align` для быстрого локального выравнивания
- `Rotate / Bake Rotation`
- `Center Origin`

## 🇬🇧 English

### What's New

- Added a full Russian user guide
- Added a full English user guide
- Added links from the main project page to both guides
- Improved GitHub-facing documentation structure
- Fixed `TO FLOOR` so the mesh is placed by its actual lowest point

### Main Features In This Version

- `ALIGN` for manual stored-side alignment
- `AUTO ALIGN` for automatic print-oriented placement
- `TO FLOOR` for moving the mesh to `Z = 0`
- `FLIP` for rotating the result by `180°`
- `Quick Align` for fast local alignment
- `Rotate / Bake Rotation`
- `Center Origin`
