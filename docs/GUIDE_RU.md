# 🇷🇺 Руководство По 3DPotok Scan Surface Align

`3DPotok Scan Surface Align` помогает быстро выравнивать сканы, hard-surface детали и сложные меши в Blender, чтобы их было удобно редактировать, подготавливать к ретопологии и укладывать для 3D-печати.

## 🧭 Где находится аддон

Панель находится здесь:

`View3D > N-panel > 3DPotok`

## 🧩 Основная логика работы

У аддона есть два основных сценария:

### 1. Ручное выравнивание по выбранным полигонам

Используйте этот режим, когда вы точно знаете, какая поверхность должна смотреть в сторону `X`, `Y` или `Z`.

Подходит для:

- выравнивания плоских частей скана
- привязки модели к понятной мировой оси
- точной ориентации детали вручную

### 2. Автоматическое выравнивание для 3D-печати

Используйте этот режим, когда нужно автоматически положить модель в удобное положение на стол или на стол печати.

Подходит для:

- подготовки меша к 3D-печати
- поиска наиболее устойчивой плоской опоры
- быстрого старта без выделения полигонов

## 🔘 Все кнопки и что они делают

### 📥 Store Selected Faces

Сохраняет выделенные полигоны как `Side 1` или `Side 2`.

Использовать:

- только в `Edit Mode`
- когда вы уже выделили нужную плоскость или группу полигонов

### 👁 Select Stored Faces

Восстанавливает ранее сохранённое выделение для стороны.

Удобно, если:

- вы хотите снова увидеть сохранённую область
- нужно повторно выровнять ту же поверхность

### 🗑 Clear Stored Faces

Очищает сохранённые полигоны для выбранной стороны.

Использовать:

- когда нужно записать другую сторону
- если вы сохранили не ту область

### 🧭 X / Y / Z

Это кнопки выбора целевой оси для стороны.

Они задают, к какой мировой оси должна быть выровнена сохранённая поверхность.

Пример:

- если хотите, чтобы выбранная плоскость стала горизонтальной, обычно выбирают `Z`
- если хотите развернуть боковую поверхность вдоль мира, используйте `X` или `Y`

### 🌐 Auto Axes

Автоматически подбирает наиболее подходящие оси для уже сохранённых сторон.

Использовать:

- после `Store Selected Faces`
- когда стороны уже выбраны, но вы не хотите вручную гадать с `X/Y/Z`

### 🧱 ALIGN

Выполняет ручное выравнивание по сохранённым сторонам и выбранным осям.

Использовать:

1. сохранить `Side 1`
2. при необходимости сохранить `Side 2`
3. выбрать оси
4. нажать `ALIGN`

### 🤖 AUTO ALIGN

Автоматически анализирует активный меш и ищет наиболее плоскую поверхность, на которую модель можно положить.

Важно:

- работает без выделения полигонов
- ориентируется на активный объект
- полезен для печати и быстрой укладки модели

Использовать:

1. выделите объект в `Object Mode`
2. нажмите `AUTO ALIGN`
3. при необходимости нажмите `FLIP`
4. затем можно нажать `TO FLOOR`

### 🔄 FLIP

Переворачивает модель на `180°` после `AUTO ALIGN`.

Использовать:

- если автоповорот в целом правильный, но модель должна смотреть в другую сторону
- если хотите быстро инвертировать положение без ручного вращения

### ⬇️ TO FLOOR

Опускает объект на пол (`Z = 0`), но работает по-разному в зависимости от режима.

В `Object Mode`:

- ищет самую нижнюю вершину объекта
- опускает меш так, чтобы именно нижняя часть легла на сетку

В `Edit Mode`:

- использует выбранную плоскость или группу выбранных полигонов
- опускает объект так, чтобы выбранная плоскость встала на `Z = 0`

Использовать:

- после `AUTO ALIGN`
- после ручного `ALIGN`
- перед экспортом
- перед проверкой укладки на стол печати

### ⚡ Quick Align

Быстро выравнивает текущее выделение в `Edit Mode`.

Кнопки:

- `Auto` - сам выбирает ближайшую ось
- `X` - выравнивает к оси `X`
- `Y` - выравнивает к оси `Y`
- `Z` - выравнивает к оси `Z`

Особенность:

- если нажать ту же ось повторно, модель перевернётся на `180°`

### 🧰 Rotate / Bake Rotation

`Rotate`:

- просто вращает объект
- сохраняет объектный rotation transform

`Bake Rotation`:

- применяет поворот к геометрии
- обнуляет rotation у объекта

Использовать `Bake Rotation`, если:

- нужно “запечь” правильное положение
- хотите экспортировать объект уже в готовой ориентации

### 🎯 Center Origin

После выравнивания переносит origin ближе к центру геометрии.

Полезно, если:

- дальше вы будете вращать объект
- нужен аккуратный pivot для сцены или экспорта

## 🛠 Рекомендуемые сценарии работы

### Вариант 1. Быстро положить скан для печати

1. Перейдите в `Object Mode`
2. Выделите объект
3. Нажмите `AUTO ALIGN`
4. Если нужно, нажмите `FLIP`
5. Нажмите `TO FLOOR`

### Вариант 2. Точно выровнять по конкретной поверхности

1. Перейдите в `Edit Mode`
2. Выделите нужные полигоны
3. Нажмите `Store Selected Faces`
4. Выберите `X`, `Y` или `Z`
5. Нажмите `ALIGN`
6. При необходимости нажмите `TO FLOOR`

### Вариант 3. Быстрые локальные повороты

1. Перейдите в `Edit Mode`
2. Выделите полигоны
3. Используйте `Quick Align`
4. Повторно нажмите ту же ось, если нужен переворот

## ⌨️ Горячие клавиши

- `Ctrl + Alt + 1` - сохранить `Side 1`
- `Ctrl + Alt + 2` - сохранить `Side 2`
- `Ctrl + Alt + X / Y / Z` - `Quick Align`
- `Ctrl + Alt + A` - `ALIGN`
- `Ctrl + Shift + Alt + A` - `AUTO ALIGN`

## 📌 Практический совет

Хороший безопасный workflow для печати:

1. `AUTO ALIGN`
2. `FLIP`, если нужно
3. `TO FLOOR`
4. при необходимости `Bake Rotation`
